# BallGame

Basic Ball Program made following instructions from J.FD

## Usage

1. put BallGame.js in a HTML document w/ an element w/ an id of 'ball'

```html
<div id="ball">@</div>
```

2. Script will detect key presses to make the ball move(left/right arrows)
